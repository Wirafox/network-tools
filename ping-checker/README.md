Narzędzie do diagnostyki sieci (Ping Checker)
Prosty skrypt w Pythonie służący do masowego sprawdzania dostępności adresów IP. Idealny do szybkiej weryfikacji listy serwerów lub urządzeń w sieci lokalnej.

Co robi skrypt?
Wczytuje listę adresów IP z pliku tekstowego.

Wykonuje test ping dla każdego adresu.

Zapisuje wyniki do pliku wynik.txt wraz z dokładną datą i godziną sprawdzenia.

Wyświetla postęp prac w konsoli w czasie rzeczywistym.

Jak używać?
Stwórz plik hosty.txt i wpisz w nim adresy IP (jeden pod drugim).

Uruchom skrypt: python skrypt.py.

Sprawdź plik wynik.txt, aby zobaczyć raport.
